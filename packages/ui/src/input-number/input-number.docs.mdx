# Input Number

`NInputNumber` is a compact numeric field with built-in increment and decrement actions. It supports min/max clamping, step values, helper text, invalid states, and accessible spinbutton semantics.

## Features

### Import

```ts
import { NInputNumber } from '@neuralangular/angular-ui/input-number';
```

### Bounded Values

Use `min`, `max`, and `stepValue` to keep numeric controls inside product limits.

### Sizes

## Theming

`NInputNumber` uses shared field tokens plus numeric-control classes: `.n-number`, `.n-number__label`, `.n-number__wrap`, `.n-number__btn`, `.n-number__input`, `.n-number__message`, `.n-number--sm`, `.n-number--md`, `.n-number--lg`, `.n-number--invalid`, and `.n-number--disabled`.

The buttons and input consume `--n-surface-2`, `--n-surface-3`, `--n-surface-4`, `--n-field-border`, `--n-field-border-focus`, `--n-focus-ring`, `--n-color-danger`, `--n-text-*`, `--n-radius-md`, and `--n-font-mono`.
