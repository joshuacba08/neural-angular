# Textarea

`NTextarea` is the multi-line companion to `NInput`. It supports labels, hints, errors, rows, size variants, resize control, and native textarea semantics.

## Theming

Classes include `.n-textarea-field`, `.n-textarea-field__label`, `.n-textarea-field__control`, `.n-textarea-field__message`, size/state modifiers, and resize modifiers for `none`, `vertical`, `horizontal`, and `both`.

`NTextarea` uses the same field tokens as `NInput`: `--n-field-bg`, `--n-field-bg-hover`, `--n-field-border`, `--n-field-border-hover`, `--n-field-border-focus`, `--n-field-placeholder`, `--n-focus-ring`, `--n-color-danger`, and the shared text, radius, spacing, and font tokens.
