### NBreadcrumbItem Interface

<ul>
  <li><code>label</code>: Text to display (optional)</li>
  <li><code>icon</code>: Icon name — <code>house</code> maps to <code>home</code> (optional)</li>
  <li><code>url</code>: Real navigation href — omit for demo/static crumbs (optional)</li>
  <li><code>disabled</code>: Disables link interaction (optional)</li>
</ul>

Items without a real <code>url</code> render as buttons (no navigation). Placeholder values like <code>#</code> are treated the same way. Use <code>(itemClick)</code> to handle crumb selection in apps or demos.

The last item is always rendered as the current page with <code>aria-current="page"</code>.
