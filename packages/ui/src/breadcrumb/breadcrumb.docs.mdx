## Features

### Import

```ts
import { NBreadcrumb } from '@neural/angular-ui/breadcrumb';
```

### Chevron Separator

The default path divider is a right chevron. Home icon support is standard.

Alternate slash divider can be combined with `activeGradient` to render the final item with Gemini gradients.

## API

### NBreadcrumb Component Inputs

<ul>
  <li><code>items</code>: Array of <code>NBreadcrumbItem</code> objects (required)</li>
  <li><code>separator</code>: Divider icon: <code>'chevron' | 'slash'</code> (default: <code>'chevron'</code>)</li>
  <li><code>activeGradient</code>: Adds Gemini text gradient to the active last item (default: <code>false</code>)</li>
</ul>

### NBreadcrumbItem Interface

<ul>
  <li><code>label</code>: Text node to show (optional)</li>
  <li><code>icon</code>: Icon name to show before text (optional)</li>
  <li><code>url</code>: Navigation anchor link path (optional)</li>
  <li><code>disabled</code>: Prevents cursor hover and link navigation (optional)</li>
</ul>

## Theming

CSS classes customize paths:

```css
/* Active last step text */
.n-breadcrumb__current {
  color: var(--n-text-1);
}

/* Link items */
.n-breadcrumb__link {
  color: var(--n-text-3);
}

/* Chevron/Slash */
.n-breadcrumb__separator {
  color: var(--n-text-4);
}
```
