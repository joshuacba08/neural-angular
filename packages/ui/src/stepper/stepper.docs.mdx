## Features

### Import

```ts
import { NStepper, NStep } from '@neural/angular-ui/stepper';
```

### Interactive Pipeline

Maintains and updates workflow indicators dynamically. Green nodes represent completed items, active items use a glowing pulse animation, and upcoming nodes are shaded.

## API

### NStepper Component Inputs

<ul>
  <li><code>activeStep</code>: 0-indexed number representing the active step card (default: <code>0</code>)</li>
</ul>

### NStep Component Inputs

<ul>
  <li><code>label</code>: Label string displayed under node (required)</li>
  <li><code>icon</code>: Icon name displayed when node is upcoming (required)</li>
  <li><code>completedIcon</code>: Icon name displayed when node is completed (default: <code>'check'</code>)</li>
  <li><code>activeIcon</code>: Icon name displayed when node is active (default: <code>'sparkles'</code>)</li>
</ul>

## Theming

Visual step skins and lines utilize CSS variables:

```css
/* completed background */
--n-color-success: #34a853;

/* active gradient circle background */
--n-gradient-primary-secondary: linear-gradient(135deg, #4285f4, #7b5cf6);

/* inactive step border color */
--n-border-2: rgba(255, 255, 255, 0.12);
```
