## API

### NStepper Inputs

<ArgTypes of={StepperStories.Playground} sort="requiredFirst" />

### NStep Inputs

<ul>
  <li><code>label</code>: Text under the node (required)</li>
  <li><code>icon</code>: Icon for upcoming steps (required)</li>
  <li><code>completedIcon</code>: Icon when step is completed (default: <code>'check'</code>)</li>
  <li><code>activeIcon</code>: Icon when step is active (default: <code>'sparkles'</code>)</li>
</ul>

Project step content inside <code>&lt;n-step&gt;</code>. Use utility classes <code>n-stepper__title</code>, <code>n-stepper__description</code>, and <code>n-stepper__actions</code> for design-system spacing.
