# Alert

`NAlert` displays inline semantic status banners. It maps to the design system's four severities (success, warning, danger, info), automatically loads default Lucide status glyphs, supports dismissible actions, and provides automatic styling wrappers for inline projected controls.

## Features

### Import

```ts
import { NAlert } from '@neural/angular-ui/alert';
```

### Basic

The default variant is `info` with a description.

<Canvas of={AlertStories.Basic} layout="centered" sourceState="shown" />

### Severities

Four semantic intents are supported out of the box, each styling borders, backdrops, text opacities, and choosing appropriate Lucide glyphs:

<Canvas of={AlertStories.Severities} layout="padded" sourceState="hidden" />

### With Actions

Project standard `<button>` tags within the alert. They will automatically be styled to match the active severity layout:

<Canvas of={AlertStories.WithActions} layout="centered" sourceState="shown" />

### Closable

Enable `closable` to render a close glyph on the right-hand side that emits the `close` output when clicked:
