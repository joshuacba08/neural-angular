# Status Dot

`NStatusDot` is a standalone Angular component that renders a small status indicator dot. It supports multiple statuses, an optional pulsing animation for active/live feedback, and fully accessible labels and semantics.

## Features

### Import

```ts
import { NStatusDot } from '@neural/angular-ui/status-dot';
```

### Basic

By default, the status is `neutral`. You can configure a text label or an accessibility label.

<Canvas of={StatusDotStories.Playground} layout="centered" sourceState="shown" />

### Statuses

All statuses apply matching semantic colors from the design system.

<Canvas of={StatusDotStories.Statuses} layout="centered" sourceState="hidden" />

### Pulse

The `pulse` property adds a subtle, expanding pulse animation around the dot, ideal for indicating active states like live streams, recording sessions, or real-time connectivity.
