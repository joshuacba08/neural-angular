# Avatar

`NAvatar` renders a profile image, derived initials, or projected content such as `NIcon`. It supports five sizes, four gradient and surface variants, optional status indicators, and overlapping stacks with `NAvatarGroup`.

## Features

### Import

```ts
import { NAvatar, NAvatarGroup } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Basic

When `src` is not provided, the component derives initials from `name`.

### Sizes & Variants

Gradient fills and the neutral surface variant stay consistent across all avatar sizes.

Provide `src` and `alt` to render a real image while keeping the same size and shape API.

### Status

Presence dots align to the bottom-right corner. `away` also softens the avatar opacity.

### Rounded Shape

Rounded squares are supported for legacy layouts, although the design reference prefers circles.

Use `NAvatarGroup` to overlap multiple participants inside a shared row.
