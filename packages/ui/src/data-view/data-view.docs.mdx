# DataView

`NDataView` is a layout container that switches display formats dynamically. It accepts structured data lists and maps them to grid cards (with custom border glows and Gemini gradients) or full-width list items using structural directive templates.

## Features

### Import

```ts
import { NDataView, NGridItem, NListItem } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Grid Layout

Display items in a multi-column card layout that scales automatically. Uses the `nGridItem` directive.

<Canvas of={DataViewStories.GridView} layout="padded" sourceState="shown" />

### List Layout

Display items in a compact, full-width vertical list stack. Uses the `nListItem` directive.

### Structural Directives

Configure separate grid and list item layout templates using direct template projection:

<div className="n-doc-table-wrap">

      </tr>
      <tr>
        <td><code>NListItem</code></td>

### Basic Setup Example

```html
<n-data-view [data]="items" [layout]="activeLayout">
  <!-- Grid Card Template -->
  <ng-template nGridItem let-item>
    <div class="n-dv-card n-dv-card--default">
      <h4>{{ item.name }}</h4>
      <p>{{ item.desc }}</p>
    </div>
  </ng-template>

  <!-- List Row Template -->
  <ng-template nListItem let-item>
    <div class="n-dv-list-item n-dv-list-item--default">
      <span>{{ item.name }}</span>
      <n-badge variant="info">{{ item.status }}</n-badge>
    </div>
  </ng-template>
</n-data-view>
```
