# Job Card

`NJobCard` represents a processing queue item from the app-pattern design system. It combines thumbnail treatment, status badge, progress shimmer, and projected workflow content such as `NAIPipeline`.

## Features

### Import

```ts
import { NJobCard } from '@neural/angular-ui/job-card';
import { NAIPipeline } from '@neural/angular-ui/ai-pipeline';
```

### Processing State

The default story follows the Oroya Video queue card: media title, model metadata, AI processing badge, progress value, and a compact horizontal `NAIPipeline` embed.

```html
<n-job-card ...>
  <n-ai-pipeline
    label="AI Pipeline"
    [steps]="jobSteps"
    orientation="horizontal"
    density="compact"
    [showProgress]="false"
    [showStatus]="false"
  />
</n-job-card>
```

## Theming

Classes include `.n-job-card`, `.n-job-card__header`, `.n-job-card__thumbnail`, `.n-job-card__badge`, `.n-job-card__progress-track`, `.n-job-card__progress-fill`, and `.n-job-card__content`.

The card consumes shared tokens for surfaces, gradient borders, semantic status colors, radius, mono text, and the primary/secondary/tertiary progress gradient.
