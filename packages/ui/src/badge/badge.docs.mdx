# Badge

`NBadge` is a compact status label for workflow states, semantic tags, counters, and highlighted metadata. It uses mono typography, two fixed heights, optional dots, and a Gemini gradient variant for high-emphasis badges.

## Features

### Import

```ts
import { NBadge } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Basic

The default badge is a neutral pill at 20px height.

### Design System

Use semantic fills for workflow states and the gradient variant for high-emphasis highlights.

### Variants

Each variant keeps the same size and silhouette API while changing tone and emphasis.

### With Dot

Enable `dot` for live states such as running, queued, or failed processes.

<Canvas of={BadgeStories.WithDot} layout="centered" sourceState="shown" />

### Sizes

Choose between 20px and 24px heights depending on density and label length.

<Canvas of={BadgeStories.Sizes} layout="centered" sourceState="hidden" />

### Shapes

Use `pill` for status chips and `square` for version tags, IDs, or compact counters.

<Canvas of={BadgeStories.Shapes} layout="centered" sourceState="shown" />

### Notification Badges (Directive)

Use the standalone `[nBadge]` directive to easily anchor absolute-positioned notification counts and status indicators to buttons or icons. The directive automatically sets `position: relative` on the host, handles offset positioning, styles count typography, and applies standard canvas outline borders to match the design spec:
