# Tree

`NTree` is a recursive expandable-node component for hierarchical data. It renders folder-like structures with animated expand/collapse, dynamic icons per state, and keyboard-friendly interaction  Eall without any `@angular/material` dependency.

## Features

### Import

```ts
import { NTree, NTreeNode } from '@neuralangular/angular-ui/tree';
```

### Design System

Hierarchical file-browser tree with animated expand/collapse transitions.

<Canvas of={TreeStories.DesignSystem} layout="padded" sourceState="shown" />

### Capabilities

<ul>
  <li><strong>Recursive rendering</strong>  Esupports infinite depth of nested nodes.</li>
  <li><strong>Animated expand/collapse</strong>  Esmooth <code>max-height</code> transitions on child containers.</li>
  <li><strong>Dynamic icons</strong>  Esupports separate <code>icon</code>, <code>expandedIcon</code> and <code>collapsedIcon</code> per node.</li>
  <li><strong>Zero dependencies</strong>  Eno <code>&#64;angular/material</code> or heavy CDK modules required.</li>
  <li><strong>SSR-safe</strong>  Eno direct browser API access.</li>
</ul>
