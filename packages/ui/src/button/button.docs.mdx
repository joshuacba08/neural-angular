# Button

`NButton` is a standalone Angular component that renders a real internal `<button>`. It supports Gemini gradient fills, tonal and ghost treatments, four sizes, icon-only mode, loading state, and full-width CTAs.

## Features

### Import

```ts
import { NButton } from '@neuralangular/angular-ui/button';
```

### Basic

The default variant is `primary`. Project content via `ng-content`.

<Canvas of={ButtonStories.Basic} layout="centered" sourceState="shown" />

### Gradient Filled

Filled buttons use directional gradients and soft glow shadows.

<Canvas of={ButtonStories.GradientFilled} layout="padded" sourceState="hidden" />

### Outlined, Tonal & Ghost

Lower-emphasis actions use gradient borders, tonal surfaces, or ghost treatments.

<Canvas of={ButtonStories.OutlinedTonalGhost} layout="padded" sourceState="hidden" />

### Sizes

Four pill heights: 32px, 40px, 48px, and 56px.

<Canvas of={ButtonStories.Sizes} layout="centered" sourceState="hidden" />

### With Icon

Project `NIcon` or any inline content before the label.

<Canvas of={ButtonStories.WithIcon} layout="centered" sourceState="shown" />

### Icon Only

Set `iconOnly` and provide `ariaLabel` for accessible naming.

<Canvas of={ButtonStories.IconButtons} layout="centered" sourceState="shown" />

### Full Width

Use `fullWidth` for page-level CTAs.

<Canvas of={ButtonStories.FullWidthCtas} layout="padded" sourceState="shown" />

### States

`loading` disables the control, sets `aria-busy`, and shows a centered spinner.
