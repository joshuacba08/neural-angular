# Select

`NSelect` is an Angular CDK overlay-backed select control with keyboard navigation, accessible combobox semantics, placeholder support, helper text, and token-based styling.

## Theming

Classes include `.n-select-field`, `.n-select-field__label`, `.n-select-field__control`, `.n-select-field__value`, `.n-select-field__panel`, `.n-select-field__option`, and size/state modifiers.

The trigger and panel consume `--n-field-border`, `--n-field-border-hover`, `--n-field-border-focus`, `--n-gradient-border`, `--n-gradient-border-subtle`, `--n-focus-ring`, `--n-elevation-4`, `--n-glow-primary-xs`, `--n-surface-2`, `--n-text-*`, and `--n-radius-*`.
