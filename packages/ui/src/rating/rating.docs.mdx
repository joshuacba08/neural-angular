# Rating

`NRating` renders an accessible star rating control for product feedback, quality scoring, and readonly summaries. It supports custom maximum values, disabled state, readonly mode, hover preview, and optional value text.

Use readonly ratings for review summaries or completed scoring states.

## Theming

Classes include `.n-rating`, `.n-rating__stars`, `.n-rating__star`, `.n-rating__star--on`, and `.n-rating__value`. The selected stars use `--n-color-warning` with a soft drop shadow, while inactive and label colors come from `--n-text-*`.

Motion respects `prefers-reduced-motion` by disabling the hover transition.
