# Card

`NCard` is the core surface primitive for grouped product content. It supports gradient-border variants, reduced-emphasis outlined containers, interactive hover lift, and a family of projected subcomponents for feature cards and horizontal row items.

## Features

### Import

```ts
import {
  NCard,
  NCardContent,
  NCardDescription,
  NCardFooter,
  NCardHeader,
  NCardIcon,
  NCardMeta,
  NCardRow,
  NCardRowAvatar,
  NCardRowBody,
  NCardRowSubtitle,
  NCardRowTitle,
  NCardRowTrailing,
  NCardTitle,
} from '@neural/angular-ui/card';
```
