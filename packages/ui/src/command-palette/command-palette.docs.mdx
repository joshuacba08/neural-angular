# Command Palette

`NCommandPalette` (also known as the Command Bar) is an interactive search-and-execute palette activated typically with `⌘K`. It provides fuzzy filtering with grouped results, keyboard navigation, execute actions, and custom shortcut key badges.

## Features

### Import

```ts
import { NCommandPalette, NCommandPaletteGroup, NCommandPaletteItem } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Basic Usage

Define groups and item options with shortcut badges, description, and status icons.

<Canvas of={CommandPaletteStories.Basic} layout="centered" sourceState="shown" />

### Keyboard Navigation

`NCommandPalette` intercepts standard arrow and enter keystrokes when focused:
- **ArrowDown**: Moves active item highlight down. Auto-scrolls list view if highlighted item overflows viewport height.
- **ArrowUp**: Moves active item highlight up.
- **Enter**: Triggers execute output on the currently active highlighted item.
- **Escape**: Triggers the `closed` event to dismiss the command palette.

### Fuzzy Search & Group Hiding

Typing in the top input filters elements by matching search queries against their label, value, and description. Groups automatically hide themselves when no matching child items exist.
