## Features

### Glow Cards

`NGlowCard` renders gradient-border cards with radial glow orbs in blue, pink, and Gemini variants.

```ts
import { NGlowCard } from '@neural/angular-ui/glow-fx';
```

<Canvas of={GlowFxStories.GlowCards} layout="padded" sourceState="shown" />

### Gradient Text Scale

Utility classes for Gemini, blue-violet, violet-pink, and white-fade gradient typography.

<Canvas of={GlowFxStories.GradientTextScale} layout="padded" sourceState="shown" />

### Neon Borders

Decorative chips with gradient borders and semantic glow shadows.

<Canvas of={GlowFxStories.NeonBorders} layout="centered" sourceState="shown" />

### Animated Gradient Rings

`NGradientRing` spins a dual-layer Gemini gradient ring. Pure CSS, SSR-safe.

```ts
import { NGradientRing } from '@neural/angular-ui/glow-fx';
```

## API

### Glow Card Inputs

<ul>
  <li><code>variant</code>: <code>'blue' | 'pink' | 'gemini'</code> (default <code>'blue'</code>)</li>
</ul>

### Gradient Ring Inputs

<ul>
  <li><code>variant</code>: <code>'gemini' | 'primary' | 'secondary' | 'cyan'</code> (default <code>'gemini'</code>)</li>
  <li><code>size</code>: CSS size string (default <code>'80px'</code>)</li>
  <li><code>duration</code>: Animation duration string (default <code>'3s'</code>)</li>
</ul>

## Theming

These components utilize CSS custom properties for styling:

```css
/* Custom variables for Glow Cards */
--n-gradient-border: linear-gradient(...);
--n-gradient-border-secondary: linear-gradient(...);

/* Custom variables for Gradient Rings */
--n-gradient-gemini: linear-gradient(...);
--n-gradient-primary-secondary: linear-gradient(...);
--n-gradient-secondary-tertiary: linear-gradient(...);
--n-gradient-primary-cyan: linear-gradient(...);
```
