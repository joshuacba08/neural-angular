# Spinner

`NSpinner` is a standalone Angular component that renders a loading indicator. It supports two modes: a circular SVG gradient ring and bouncing loading dots, as well as multiple sizes, color variants, and accessible screen-reader labels.

## Features

### Import

```ts
import { NSpinner } from '@neural/angular-ui/spinner';
```

### Basic

By default, the spinner renders in `ring` mode with `md` size and `primary` variant.

<Canvas of={SpinnerStories.Playground} layout="centered" sourceState="shown" />

### Modes

Choose between a circular SVG gradient `ring` and bouncing loading `dots`.

<Canvas of={SpinnerStories.Modes} layout="centered" sourceState="hidden" />

### Sizes

Three pre-configured sizes: `sm` (28px), `md` (40px), and `lg` (40px).

<Canvas of={SpinnerStories.Sizes} layout="centered" sourceState="hidden" />

### Variants

Four color variants mapping to theme colors: `primary`, `neutral`, `success`, and `danger`.
