# OrgChart

`NOrgChart` is a recursive organizational and pipeline hierarchy display. It automatically draws connection lines and handles horizontal (pipeline) or vertical tree arrangements based on individual node configuration.

### Design System

Represents the exact hierarchy layout specified in the Design System.

<Canvas of={OrgChartStories.DesignSystem} layout="padded" sourceState="shown" />

### Capabilities

<ul>
  <li><strong>Recursive Rendering</strong> — renders nested node hierarchies at any depth.</li>
  <li><strong>Layout Flexibility</strong> — nodes support horizontal (pipeline columns) or vertical lists under them.</li>
  <li><strong>Style Customization</strong> — supports inline styles and label classes for color coding pipelines.</li>
  <li><strong>Spacing controls</strong> — tune <code>boxSpacing</code>, <code>levelGap</code>, and <code>childGap</code> (px).</li>
  <li><strong>Tree connectors</strong> — continuous T-junction rails via pseudo-elements (no fixed-width segments).</li>
</ul>
