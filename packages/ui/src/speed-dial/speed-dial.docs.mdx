# SpeedDial

`NSpeedDial` is a floating action button (FAB) that reveals staggered action items on click. Matches **Actions & Menus · 01** from the Neural design system.

## Features

### Import

```ts
import { NSpeedDial, NSpeedDialItem } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Design System Overview

Three layout variants side by side: linear stack (open), semi-circle fan, and click-to-open.

<Canvas of={SpeedDialStories.DesignSystem} layout="fullscreen" sourceState="shown" />

### Linear Stacking

Space items vertically (or horizontally) from the FAB with an animated stagger delay.

<Canvas of={SpeedDialStories.LinearUp} layout="centered" sourceState="shown" />

### Semi-circle Fan

Items spread in a horizontal row above the FAB  Ethe design system's "Semi-circle · Up" layout.

<Canvas of={SpeedDialStories.SemiCircleUp} layout="centered" sourceState="shown" />

### Radial Arc

Optional circular arc layout for denser radial menus.

<Canvas of={SpeedDialStories.RadialUp} layout="centered" sourceState="shown" />
