# Advanced Forms

Advanced Forms is the combined showcase for Neural Angular form primitives that are designed to work together in dense application surfaces: float labels, icon fields, grouped inputs, masks, pickers, select controls, toggles, and split actions.

## Theming

Use Advanced Forms as a composition reference rather than a single install target. The primitives stay independent so applications can import only the controls they need.

Prefer native inputs inside `NFloatLabel`, `NIconField`, and `NInputGroup` when a full Angular component would be too heavy. Reach for dedicated components like `NSelect`, `NPassword`, `NInputOtp`, or `NRating` when behavior and accessibility are part of the control.
