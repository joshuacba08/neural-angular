## Download

Install the package and its peer dependencies from the npm registry.

```bash
# Using npm
npm install @neuralangular/angular-ui @angular/cdk @lucide/angular

# Using yarn
yarn add @neuralangular/angular-ui @angular/cdk @lucide/angular

# Using pnpm
pnpm add @neuralangular/angular-ui @angular/cdk @lucide/angular
```

## Styles

Import the design-system stylesheet once in your global application styles.

```css
@import "@neuralangular/angular-ui/styles";
```

The bundle includes design tokens, dark and light theme layers, and the shared base treatment
 for typography and canvas surfaces.

## Fonts

The default typography tokens target **Plus Jakarta Sans** and **JetBrains Mono**. Load them in
your app shell for the intended appearance.

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@300;400;500;600;700;800&family=JetBrains+Mono:wght@400;500;600&display=swap"
  rel="stylesheet"
/>
```

## Verify

Verify your setup by importing components from the root entry point.

```typescript
import { Component } from '@angular/core';

import {
  NButton,
  NCard,
  NCardContent,
  NCardHeader,
  NCardTitle,
} from '@neuralangular/angular-ui';

@Component({
  selector: 'app-root',
  standalone: true,
  imports: [NButton, NCard, NCardContent, NCardHeader, NCardTitle],
  template: `
    <n-card>
      <n-card-header>
        <n-card-title>Neural UI ready</n-card-title>
      </n-card-header>

      <n-card-content>
        <n-button variant="primary">Primary action</n-button>
      </n-card-content>
    </n-card>
  `,
})
export class AppComponent {}
```

> **v0.1.x note** — All components are exported from the single root entry `@neuralangular/angular-ui`.
> Granular per-component entry points (`/button`, `/card`, etc.) are planned for a future minor release.
