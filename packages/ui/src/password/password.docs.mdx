# Password

`NPassword` is a secure text field with a built-in visibility toggle and optional strength meter. It is designed for auth screens, settings panels, and account forms.

## Theming

Classes include `.n-password`, `.n-password__label`, `.n-password__shell`, `.n-password__control`, `.n-password__toggle`, `.n-password__strength`, `.n-password__bar`, `.n-password__strength-label`, `.n-password__message`, and the size/state modifiers.

The strength meter uses `--n-color-danger`, `--n-color-warning`, and `--n-color-success`. The field shell uses the shared field tokens, standard text tokens, `--n-radius-md`, and `--n-focus-ring`.
