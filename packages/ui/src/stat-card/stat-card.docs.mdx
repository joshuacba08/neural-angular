# Stat Card

`NStatCard` is a standalone Angular component designed to showcase key metrics, statistics, and KPI dashboards. It supports multiple sizes, border gradient variants, icon attachments, semantic trend flags, and interactive hover effects.

## Features

### Import

```ts
import { NStatCard } from '@neural/angular-ui/stat-card';
```

Default layout and dimensions, with interactive hover effects enabled.
