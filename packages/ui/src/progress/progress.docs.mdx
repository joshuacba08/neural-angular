# Progress

`NProgress` and `NProgressRing` are standalone Angular components used to indicate the status of an ongoing process. They support linear, segmented, circular, and advanced "orbital" layouts with Gemini gradients, pulsing shimmers, and dark-mode optimization.

## Features

### Import

```ts
import { NProgress, NProgressRing } from '@neural/angular-ui/progress';
```

### Basic Linear Progress

Linear progress indicators occupy the full width of their container and can show determinate status with a percentage value.

<Canvas of={ProgressStories.Basic} layout="centered" sourceState="shown" />

### Variants

Choose between 9 standard brand/semantic color variants.

<Canvas of={ProgressStories.Variants} layout="centered" sourceState="hidden" />

### Sizes

Four pre-configured sizes matching layout heights: `xs` (4px), `sm` (5px), `md` (9px), and `lg` (10px).

<Canvas of={ProgressStories.Sizes} layout="centered" sourceState="hidden" />

### Indeterminate

Use `indeterminate` mode when the total time or steps of an operation is unknown. It supports two modes: `fill` (sliding gradient) and `bar` (bouncing bar segment).

<Canvas of={ProgressStories.Indeterminate} layout="centered" sourceState="hidden" />

### Segmented Progress

Segmented progress is activated by providing `totalSteps`. It breaks the bar into separate indicators, showing progress in steps rather than a continuous line.

<Canvas of={ProgressStories.Segmented} layout="centered" sourceState="hidden" />

### Progress Rings & Orbital AI

`NProgressRing` provides circular loading indicators. In `orbital` size, it displays an overlay containing a center label, a metric description, and an sub-caption (ideal for GPU/CPU task status cards).
