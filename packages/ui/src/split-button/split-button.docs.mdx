# Split Button

`NSplitButton` is a standalone Angular component for pairing one primary action with a compact dropdown of related alternatives. It supports primary and neutral variants, optional leading icon, menu item icons, separators, and outside-click dismissal.

## Features

### Import

```ts
import { NSplitButton } from '@neuralangular/angular-ui/split-button';
```

### Basic

The main button emits `click`; the caret opens a menu built from `model`.

<Canvas of={SplitButtonStories.Basic} layout="centered" sourceState="shown" />

### Variants

Use `primary` for high-emphasis commit actions and `neutral` for secondary grouped actions.

<Canvas of={SplitButtonStories.Variants} layout="centered" sourceState="hidden" />

### With Icon

The root action can show a leading `NIcon`, and menu items can include per-item icons.

<Canvas of={SplitButtonStories.WithIcon} layout="centered" sourceState="shown" />

### Menu Model

Menu items support icons, semantic icon colors, separators, and per-item commands.
