# Icon

`NIcon` is the shared glyph primitive for the Neural UI system. It wraps Lucide with a typed registry, token-based sizing, `currentColor` inheritance, and a provider-first setup that stays friendly to Angular SSR.

## Features

### Import

```ts
import { NIcon } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Basic

The component ships with a typed `name` contract, predictable sizing, and an automatic accessible label fallback when `decorative` is disabled.

<Canvas of={IconStories.Basic} layout="centered" sourceState="shown" />

### Scale

Every icon size maps to shared design tokens, keeping buttons, chips, badges, avatars, and cards visually aligned.

<Canvas of={IconStories.SizeScale} layout="padded" sourceState="hidden" />

### Stroke Control

Lucide stroke weight can be tuned per use case without introducing alternate icon assets.

<Canvas of={IconStories.StrokeWeights} layout="padded" sourceState="hidden" />

### Semantic Color

Icons inherit `currentColor`, so the same glyph adapts naturally across status, emphasis, and destructive surfaces.

### In Context

## Registry & Provider

### Provider Setup

Register the curated Neural icon pack once at the app boundary.

```ts
import { provideNeuralIcons, provideNeuralTheme } from '@neuralangular/angular-ui';

export const appConfig = {
  providers: [
    provideNeuralTheme({ defaultTheme: 'dark' }),
    provideNeuralIcons(),
  ],
};
```
