# Input

`NInput` is the standard text field for Neural Angular forms. It supports labels, hints, errors, three sizes, visual variants, and native input types while keeping the DOM simple and SSR-safe.

## Features

### Import

```ts
import { NInput } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Text Inputs

Use `label`, `placeholder`, `hint`, and `error` for the common field states.

## Theming

`NInput` uses the shared field token set: `--n-field-bg`, `--n-field-bg-hover`, `--n-field-border`, `--n-field-border-hover`, `--n-field-border-focus`, `--n-field-placeholder`, `--n-focus-ring`, `--n-color-danger`, and the standard `--n-text-*`, radius, spacing, and font tokens.

Root classes include `.n-field`, `.n-field__label`, `.n-field__control`, `.n-field__message`, `.n-field--sm`, `.n-field--md`, `.n-field--lg`, `.n-field--filled`, `.n-field--ghost`, `.n-field--invalid`, and `.n-field--disabled`.
