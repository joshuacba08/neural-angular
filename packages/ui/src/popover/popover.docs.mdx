## Features

### Glass Panels & Tooltips

The reference layout combines a glass info panel, directional tooltip, and context menu from the design system.

<Canvas of={PopoverStories.DesignSystem} layout="centered" sourceState="shown" />

### Popover basic

Use `nPopover` to attach a glass-styled text message or template overlay on click, hover, or focus.

```ts
import { NPopoverDirective } from '@neural/angular-ui/popover';
```

<Canvas of={PopoverStories.PopoverBasic} layout="centered" sourceState="hidden" />

### Popover Template

Pass an Angular `<ng-template>` to render rich floating panel content.

<Canvas of={PopoverStories.PopoverTemplate} layout="centered" sourceState="hidden" />

### Tooltips & Directional Arrows

Tooltips display quick text indicators on hover/focus with surface-4 styling and positional arrows.

```ts
import { NTooltipDirective } from '@neural/angular-ui/tooltip';
```

<Canvas of={PopoverStories.Tooltips} layout="centered" sourceState="hidden" />

### Glass Panel

Use `.n-glass-panel` for gradient-border glass surfaces with backdrop blur.

<Canvas of={PopoverStories.GlassPanel} layout="centered" sourceState="hidden" />

### Context Menu

Use `.n-ctx-menu` and `.n-ctx-item` utility classes for contextual action menus.

## API

The suite exposes two main directives: `NPopoverDirective` and `NTooltipDirective`.
