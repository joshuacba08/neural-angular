## API

The suite exposes two main directives: `NPopoverDirective` and `NTooltipDirective`.

### Popover Directive API

<ArgTypes of={PopoverStories.PopoverBasic} sort="requiredFirst" />

### Tooltip Directive API

The `NTooltipDirective` accepts:
<ul>
  <li><code>nTooltip</code>: The string to display inside the tooltip.</li>
  <li><code>nTooltipPosition</code>: <code>'top' | 'bottom' | 'left' | 'right'</code> (default <code>'top'</code>)</li>
  <li><code>nTooltipDisabled</code>: Boolean to temporarily disable the tooltip.</li>
  <li><code>nTooltipShowDelay</code>: Transition delay before showing tooltip (default <code>120</code>ms)</li>
  <li><code>nTooltipHideDelay</code>: Transition delay before hiding tooltip (default <code>80</code>ms)</li>
</ul>

## Theming

These components consume CSS custom properties for background colors, borders, and shadows:

```css
/* Tooltip properties */
--n-tooltip-bg: var(--n-surface-3);
--n-tooltip-border: var(--n-border-1);

/* Popover properties */
--n-gradient-border-primary: linear-gradient(...);

/* Glass panel */
.n-glass-panel

/* Context menu */
.n-ctx-menu
.n-ctx-item
.n-ctx-item--active
.n-ctx-item--danger
```
