# Chip

`NChip` supports two design-system roles: selectable filter chips and semantic tag chips. It also supports compact density, removable tags, and rich projected content such as icons or avatars.

## Features

### Import

```ts
import { NChip } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Basic

The default chip is a medium filter pill.

<Canvas of={ChipStories.Basic} layout="centered" sourceState="shown" />

### Design System

Filter chips, semantic tags, removable tags, and avatar chips all share the same core primitive.

### Filter Chips

Use filter mode for selectable catalog, model, or workflow toggles.

### Tag Variants

Tag mode shifts into semantic inline labels with severity color treatment.

<Canvas of={ChipStories.TagVariants} layout="centered" sourceState="hidden" />

### Removable

When `removable` is enabled the close affordance emits the `removed` output.

<Canvas of={ChipStories.Removable} layout="centered" sourceState="shown" />

### Status tag recipes

You can project custom status dots (optionally pulsing) inside `NChip` to build semantic live tags:

```html
<div style="display:flex;gap:8px">
  <n-chip mode="tag" variant="success">
    <span style="width:6px;height:6px;border-radius:50%;background:var(--n-color-success)"></span>
    Processing
  </n-chip>
  <n-chip mode="tag" variant="primary">
    <span style="width:6px;height:6px;border-radius:50%;background:var(--n-color-primary);animation:n-status-dot-pulse 1.5s ease-in-out infinite"></span>
    Enhancing
  </n-chip>
</div>
```
