# Data Card

`NDataCard` is a premium container surface designed to show key entity metadata, lists of key-value rows, or embedded custom SVG charts (such as bar charts, donut charts, and sparklines) without external graphing dependencies.

## Features

### Import

```ts
import { NDataCard, NBarChart, NDonutChart, NSparkline, NECharts } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Basic

Display simple lists of label-value elements with status indicators.

<Canvas of={DataCardStories.Basic} layout="centered" sourceState="shown" />

### Bar Chart (GPU History)

Supports fluctuating SVG bars inside a gradient card wrapper.

<Canvas of={DataCardStories.BarChartGpuHistory} layout="centered" sourceState="shown" />

### Donut Chart (VRAM Distribution)

Draws stacked circle segments with customizable percentages and inline legend keys.

<Canvas of={DataCardStories.DonutVramDistribution} layout="centered" sourceState="shown" />

### Sparkline Metrics

Premium inline SVG trendlines indicating value changes over time.

<Canvas of={DataCardStories.SparklineMetrics} layout="padded" sourceState="shown" />

### Interactive Chart (Apache ECharts)

A lazy-loaded, SSR-safe ECharts wrapper that automatically resizes and updates theme styling on dark/light changes.
