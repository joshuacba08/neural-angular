# Table

`NTable` is a high-performance, design-system-aligned table component. It supports interactive client-side sorting, row selection (single or multiple checkboxes), and a paginator footer with dynamic rows-per-page selection.

### Custom Cell Templates

To render HTML (like status badges or links) instead of plain text, project an `<ng-template>` with `nTemplate` structural directive mapping to the column's `key`:

```html
<n-table [columns]="columns" [data]="data">
  <ng-template nTemplate="status" let-value let-row="row">
    <n-badge [variant]="value === 'Done' ? 'success' : 'info'">
      {{ value }}
    </n-badge>
  </ng-template>
</n-table>
```
