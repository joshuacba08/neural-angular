# PickList

`NPickList` is a dual-panel list control that lets users select and transfer items between a source and a target list. It supports drag-and-drop, multi-selection for bulk transfers, and emits array updates for two-way binding.

## Features

### Import

```ts
import { NPickList, NPickListItem } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Capabilities

<ul>
  <li><strong>Drag &amp; Drop</strong>  Epowered by Angular CDK <code>DragDropModule</code>.</li>
  <li><strong>Multi-Selection Transfers</strong>  Eselect multiple items in either panel and transfer them in bulk.</li>
  <li><strong>State Synchronization</strong>  Eemits <code>sourceChange</code> and <code>targetChange</code> for two-way binding.</li>
  <li><strong>Design System Layout</strong>  Egrid with <code>540px</code> max width, <code>10px</code> column gap, and <code>30px</code> transfer buttons.</li>
</ul>
