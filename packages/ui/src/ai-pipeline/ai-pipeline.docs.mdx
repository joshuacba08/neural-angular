# AI Pipeline

`NAIPipeline` documents multi-step AI processing flows from the design system. It supports vertical inspector timelines, compact horizontal job-card pipelines, semantic step states, and optional progress.

## Theming

The pipeline uses `.n-ai-pipeline`, `.n-ai-pipeline-step`, state modifiers like `.n-ai-pipeline-step--running`, and orientation modifiers for vertical or horizontal layouts.

It consumes shared Neural tokens including `--n-color-primary-bright`, `--n-color-success`, `--n-color-warning`, `--n-color-danger`, `--n-pipeline-line`, `--n-pipeline-step-bg`, `--n-border-1`, `--n-surface-2`, and spacing/radius tokens.
