## Features

### Import

```ts
import { NTimeline, NTimelineItem } from '@neuralangular/angular-ui';
```

> **v0.1.x** — Import from the root entry `@neuralangular/angular-ui`. Per-component subpath imports are planned for a future release.

### Processing Log

The default timeline uses timestamps, status colors, active running state, and pending tail events.

<Canvas of={TimelineStories.Default} layout="padded" sourceState="shown" />

### Activity Feed

Activity feeds use inline timestamps, icons, composed badges, progress, and actions through the item slot.
